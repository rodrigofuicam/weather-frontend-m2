# ClimaSur

ClimaSur es una aplicación web que muestra el clima actual y la predicción semanal de ciudades de la región de Los Lagos, Chile. Ideal para planificar tu día y la semana según el clima.

---

## Características

- Consulta del clima actual por ciudad.
- Predicción semanal: temperatura, humedad y viento.
- Iconos representativos del estado del clima.
- Diseño responsivo para móvil y escritorio.

---

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge, Safari)
- Librerías: [Bootstrap 5](https://getbootstrap.com/), [jQuery](https://jquery.com/)
